# 📁 Drag & Drop Import - Quick Guide

**Access:** Sidebar → "📁 Import Documents"  
**URL:** https://docs.bitandmortar.com/Import_Documents

---

## ✅ Supported Formats

**PDF** • **MD** • **TXT** • **DOCX** • **PAGES** ⭐ • **RTF** • **ODT**

---

## 🎯 How to Use

### 1. Drag & Drop
Drag files onto upload zone → Auto-converts → Auto-indexes

### 2. Or Browse
Click to select files → Upload → Indexed in ~5 seconds

### 3. Bulk Import
Enter folder path → Click "Import Folder" → All files indexed

---

## 🍎 Apple Pages Support

`.pages` files automatically:
1. Unzipped
2. Preview extracted
3. Converted to Markdown
4. Indexed like normal docs

---

## 📊 What Happens

```
Upload → Convert → Chunk → Embed → Index → Ready!
  ↓        ↓         ↓        ↓        ↓       ↓
  File   Markdown  500w    Vectors  LanceDB  Search
```
